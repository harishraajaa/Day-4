1. Spread Operator


Converts a array into comma values.

[1,2,3,5] into 1,2,3,5


2. Rest Operator

Used in Arguments of a function.( while reciving a input)

example:

funciton Display(...input)
{
	Console.log(input)
}


Diff b/w Spread and Rest

Spread : used while sending the input data to a function

Rest: Used while receiving a input date in a function (Arguments)
